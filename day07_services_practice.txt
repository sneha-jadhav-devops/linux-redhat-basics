3. Start nginx service:
sudo systemctl start nginx

4. Check nginx status:
systemctl status nginx

5. Enable nginx on boot:
sudo systemctl enable nginx

6. Restart nginx:
sudo systemctl restart nginx

7. View nginx logs:
journalctl -u nginx
